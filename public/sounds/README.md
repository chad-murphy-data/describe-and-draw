# Sound Effects for Blind Draw

Download these free sound effects and place them in this folder.

## Required Files

| File | Description | Suggested Source |
|------|-------------|------------------|
| `curtain.mp3` | Fabric swoosh for curtain reveal | [Mixkit - Swoosh](https://mixkit.co/free-sound-effects/swoosh/) |
| `drumroll.mp3` | Drumroll for score anticipation | [Mixkit - Drum](https://mixkit.co/free-sound-effects/drum/) |
| `ding.mp3` | Chime when score locks in | [Mixkit - Notification](https://mixkit.co/free-sound-effects/notification/) |
| `whoosh.mp3` | Quick transition sound | [Mixkit - Swoosh](https://mixkit.co/free-sound-effects/swoosh/) |
| `celebration.mp3` | Small win sound for awards | [Mixkit - Win](https://mixkit.co/free-sound-effects/win/) |
| `fanfare.mp3` | Victory fanfare for winners | [Mixkit - Win](https://mixkit.co/free-sound-effects/win/) |

## How to Download

1. Visit the Mixkit links above
2. Find a suitable sound effect
3. Click "Download Free SFX"
4. Rename the file to match the names above
5. Place in this `public/sounds/` folder

The app will work without sounds - they're optional enhancements.
